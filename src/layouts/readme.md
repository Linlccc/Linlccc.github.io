# 布局目录文件解释

## 标记解释

``` text
"#"   表示已修改
"@"   完全自己写
```

## layouts

``` text
根目录
```

- 404.html
  - 404页面模板
- rebots.txt[@]
  - robots.txt 爬虫配置文件

``` text

```

- list.html
